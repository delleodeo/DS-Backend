## Summary

This system ensures vendor monthly revenues are automatically tracked and historicized without any manual intervention. The `currentMonthlyRevenue` acts as a running total for the current month, which gets pushed to the historical `monthlyRevenueComparison` array at month's end and then reset for the new month.
